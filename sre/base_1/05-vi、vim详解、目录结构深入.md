## 网卡配置文件
```shell
/etc 所有配置目录
/etc/sysconfig 系统配置目录
/etc/sysconfig/network-scripts 网络配置目录
/etc/sysconfig/network-scripts/ifcfg-eth0 网卡配置文件

重启网卡
systemctl restart network
```
