# systemctl
```bash
#设置开机自启动
systemctl enable sshd

#设置开机不自动启动
systemctl disable sshd

#设置当前运行
systemctl start sshd || systemctl restart sshd

#关闭防火墙
systemctl stop firewalld.service

#检查状态
systemctl status firewalld.service

```
