## bash模式
```bash
1.比较两个文件差异diff
vim -d file1 file2
2.打开两个文件并分屏
vim -o file1 ile2 
```
## 插入模式
```bash
1.i
在光标处插入
2.o
在下一行插入
3.a
在光标后插入
4.O
在上一行插入
5.A
在行尾插入
6.I
在行尾插入
```
## 扩展模式
```bash
:
 q
 wq
 %s#word#word2#g 全局替换
 /word            查找
 set nu        
 set nonu
 syntax off | on
 r filename 读另一个文件到当前文件中

```
## 命令模式
```bash
移动操作
1.G 一到最后一行
2.1G | gg 移到第一行
3.0 移到行首
4.$ 移到行尾
5.^ 有空格，移到首字符

字符操作
6.x 删除光标处的字符
7.xp 交换光标所在处的字符和其后面字符的位置
8.g~~ 转换大小写，一整行
9.~ 转换大小写

替换操作
10.r 替换光标处的字符
11.R 连续替换模式

删除操作
12.5dd 删除5行
13.d^ 删除到行首
14.d$ 删除到行尾
15.D  删除到行尾

复制操作
16.yy 复制行
17.y^
18.y$

粘贴命令
19.p 

分屏操作
20.:vsplit 水平
21.:split 垂直
22.ctrl + w 方向 切换分屏
23.ctrl + w o 取消所有分屏
 
```
