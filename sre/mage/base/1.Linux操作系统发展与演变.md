## 计算机组成
```shell
1.计算机的组成由五大部分组成，遵循冯诺依曼体系
	CPU(运算器、控制器、寄存器、缓存)
		CPU内部自带指令集(普通指令集、特权指令集)
		运算器带有运算指令集
		控制器带有控制指令集
	存储器(内存：RAM Random Access Memory 随机访问内存)
	Input|(输入设备)下指令、提供数据等
	Output(输出设备)输出数据加工后的结果
	
2.程序组成
	指令+数据 
		数据存储在内存中，最小存储单元是一个字节Bytes八位bit
	算法+数据结构
	
3.CPU和内存如何联系
	32位CPU有32根传输线，寻址地址为2^32次方，大约4G
	数据存储在内存中，而数据处理在CPU中，所以要把数据从内存往CPU运送
	程序开始执行，CPU从内存中读取指令存到寄存器，再根据指令去读取对应的数据也存到寄存器。在读取第一条指令后，不会等待第一条指令的数据读取完成，而是会继续读取下一条指令，直到某一条指令的数据完全读取后，再开始执行某一条指令。CPU工作是并行、异步+动态调度的。简单来讲就是一直在读取，不会等待。
	
4.多任务
	早期的计算机由于CPU和内存以及输入输出设备的速度相差太大，而且只能同一时间只运行一个程序，这就造成了CPU的利用率太低，大部分时间都处于等待空闲状态。
	为什么只能同一时间运行一个程序呢？因为多个程序同时存储在一条内存条上，会造成冲突覆盖，最后所有程序都不能运行。
	这时就有三家公司贝尔公司BELL.麻省理工MIT.GE电器公司三家公司研究出了Multics多理念，该理念对内存条进行空间划分，规定一个程序只能待在一块空间内，互不干扰。CPU则进行时间切割，轮转处理每一个程序的内容
	
5.操作系统的功能
	驱动程序：调动硬件工作
	进程管理：多任务，监控进程互不干扰
	内存管理：多任务，给进程分配内存
	文件系统
	网络功能
	安全
```
